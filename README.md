# 💈 AI Hairstyle Recommendation System

An AI-powered web application that recommends hairstyles based on face shape using a Convolutional Neural Network (CNN) and allows users to preview hairstyles through AI Hair Try-On.

---

## 📌 Features

- 🔍 Face Shape Detection using CNN
- 💇 Hairstyle Recommendation
- 🎨 AI Hair Try-On
- 🤖 AI Hairstyle Consultant
- 📚 Hairstyle Metadata Management
- 📊 Scan History
- 👤 Admin Dashboard

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/USERNAME/barbershop-ai.git
```

---

### Backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
```

---

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## Environment Variables

Create a `.env` file inside the backend folder.

```env
REPLICATE_API_TOKEN=YOUR_REPLICATE_API_TOKEN
DATABASE_URL=YOUR_DATABASE_URL
```
